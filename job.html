Sr. Front-End Developer

See more roles Learn about us

Company> Careers> Sr. Front-End Developer

We are seeking a skilled front-end developer with a passion for crafting top-notch web applications. If you have expertise in React.js development and are enthusiastic about leveraging the latest technologies to create outstanding user experiences do not hesitate to apply.

Responsibilities

  • Web Development: Collaborate with the team to develop and maintain responsive web applications using HTML, CSS, JavaScript, and TypeScrip
  • React.js Development: Assist in building and enhancing user interfaces with React.js, ensuring high performance and scalability
  • Learning and Growth: Stay up-to-date with the latest frontend technologies and trends, actively seeking opportunities for skill enhancement
  • API Integration: Integrating with (Backend and 3rd party) APIs
  • Mentorship: Provide mentorship and guidance to junior developers, fostering a collaborative and knowledge-sharing environment within the team
  • Performance Optimization: Identify and address performance bottlenecks in the application, optimizing for speed and responsiveness
  • Code Documentation: Document codebase effectively, providing clear and concise documentation for fellow developers and future reference

Qualifications

  • BS/MS degree in Computer Science, Engineering or a related subject
  • Proven experience of at least 5 years working as a Front-End Developer
  • High proficiency with web technologies, including HTML, CSS, and JavaScrip
  • Experience with React JS and TypeScript is a must
  • Experience with state management libraries (e.g., Redux, Context API)
  • Proficient understanding of code versioning tools, such as Git
  • Excellent communication and teamwork skills
  • Eagerness to learn and adapt in a dynamic and collaborative environment

What We Offer

  • Mentorship and guidance from experienced developers
  • Opportunities for professional growth and skill development
  • A supportive and inclusive work environment
  • Exciting projects that challenge and inspire
